# 프로젝트 - 중계나라🥕
- 중고거래 사이트를 벤치마크 하여 구매자와 판매자를 연결하여 중계해주는 중고거래 챗봇을 만들었습니다.
---
## 🔍 개발 스팩
- 프론트엔드: 카카오 챗봇 (Json 통신)
- 백엔드:
  - 언어: Java 11
  - 프레임워크: Spring boot 2.7
  - 데이터베이스: Mysql 8.0
  - ORM: JPA (Hibernate)
  - 주요 라이브러리: Redis, JWT, Jsoup
  - 테스트 라이브러리: Junit, MockMvc 
  - AWS: Ec2, S3, Route 53
  - CI/CD: GitHub Action, Docker
